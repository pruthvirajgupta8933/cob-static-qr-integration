Quick Integration

1
Get API Credentials
generateAPIKeys('MERCH_ID', 'Name')
2
Install Client SDK
npm install sabpaisa-sdk
3
Initialize Client
const client = new SabPaisaQRClient(
  apiKey, apiSecret, 'production'
);
4
Make API Calls
const qr = await client.generateQR({
  merchant_name: 'Store',
  amount: 500
});
